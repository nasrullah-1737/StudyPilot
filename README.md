 📱 StudyPilot – Student Helper App

StudyPilot  is a modern, all-in-one student productivity application built using NET MAUI and C# , designed to help students manage their academic life efficiently. The app combines essential tools like notes, task management, timetable scheduling, and focus tracking into a single, clean, and user-friendly interface.


 🚀 Features

 🔐 Authentication

* User login and registration
* Secure session handling (local storage)

 🏠 Dashboard

* Overview of daily tasks and schedule
* Summary cards for pending and completed tasks
* Quick access to important features

 📝 Notes Management

* Create, edit, and delete notes
* Support for text and image-based notes
* Organize notes by subjects
* Local data storage using SQLite

 ✅ Task & Assignment Manager

* Add tasks with title, description, deadline, and priority
* Mark tasks as completed
* Filter tasks (All / Completed / Pending)

 📅 Timetable

* Weekly class schedule
* Add and manage subjects with time slots
* View today's classes at a glance

 ⏰ Reminders & Notifications

* Local notifications for deadlines and classes

 🍅 Focus Mode (Pomodoro Timer)

* 25-minute focus sessions with 5-minute breaks
* Start, pause, and reset functionality
* Visual progress tracking

 📂 File Storage

* Upload and view study materials (PDFs, images)
* Local file management


 🛠️ Tech Stack

 🎨 UI/UX

The goal of StudyPilot is to simplify student life by providing a centralized platform for managing academic tasks, improving productivity, and enhancing focus.
